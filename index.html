How it works: Your trail data syncs to a private GitHub repo (only you can see it). This app's HTML is hosted separately on a public repo via GitHub Pages.

Setup (one time):
1. Create a private repo (e.g. trail-data) — this stores your data
2. Create a public repo (e.g. trail-journal) — put this HTML file as index.html, enable Pages
3. Create a fine-grained token with access to the private repo only
4. Enter the private repo details below

Token setup: GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token
• Repository access → Only select repositories → pick your private data repo
• Permissions → Contents: Read and write
• The token never leaves your browser. Each device needs its own token entry.
